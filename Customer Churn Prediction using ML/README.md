# Customer Churn Prediction using Machine Learning

## Overview
This project predicts whether a customer is likely to churn using historical customer data.

## Dataset
- Telco Customer Churn Dataset
- Contains customer demographics, services used, and account information

## Approach
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Feature encoding and scaling
- Model training and evaluation

## Models Used
- Logistic Regression
- Random Forest (optional)

## Evaluation
- Accuracy
- Confusion Matrix

## Tools & Technologies
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib / Seaborn

## Key Learnings
- Importance of feature engineering
- Handling categorical variables
- Interpreting model performance
