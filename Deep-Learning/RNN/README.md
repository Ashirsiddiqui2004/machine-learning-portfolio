# RNN Sentiment Analysis

This project demonstrates sentiment analysis using a Recurrent Neural Network (RNN).

## Project Overview
- Text sentiment classification
- Sequence modeling with RNN
- Focus on understanding sequential data
- Implemented using TensorFlow / Keras

## Tech Stack
- Python
- TensorFlow / Keras
- NumPy

## Notes
- Educational project for learning RNN concepts
- Forms the base for LSTM improvements
