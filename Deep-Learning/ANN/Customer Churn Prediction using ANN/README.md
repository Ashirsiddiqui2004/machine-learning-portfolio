# Customer Churn Prediction using ANN

This project uses an Artificial Neural Network (ANN) to predict whether a customer
will churn based on historical data.

## Problem Type
- Binary Classification

## Model Used
- Artificial Neural Network (ANN)

## Workflow
- Data preprocessing
- Feature scaling
- ANN model training
- Model evaluation

## Tools
- Python
- TensorFlow / Keras
- Pandas, NumPy
