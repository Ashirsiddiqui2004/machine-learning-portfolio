# Handwritten Digit Classification using ANN

This project focuses on classifying handwritten digits using an Artificial Neural Network.

## Dataset
- MNIST Handwritten Digits Dataset

## Problem Type
- Multi-class Classification

## Model Used
- Artificial Neural Network (ANN)

## Key Steps
- Data normalization
- ANN training
- Model evaluation

## Tools
- Python
- TensorFlow / Keras
- NumPy, Matplotlib
