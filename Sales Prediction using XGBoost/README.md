# Sales Prediction using XGBoost

This project focuses on predicting sales using machine learning techniques,
with XGBoost as the primary model.

## Problem Statement
Accurate sales prediction helps businesses in demand planning, inventory
management, and revenue forecasting.

## Dataset
- Historical sales data
- Contains features related to product, pricing, and sales volume

## Approach
1. Data cleaning and preprocessing
2. Feature engineering
3. Train-test split
4. Model training using XGBoost
5. Model evaluation

## Model Used
- XGBoost Regressor
